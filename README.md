## Example

Easy to add target Example:
```swift
btn.touchUpInside {
    print("action")
}
btn.touchDown {
    print("action")
}
```

## Requirements

swift 4.2
Xcode 10.0

## Installation

### Using [CocoaPods](https://cocoapods.org).
Add the following line to your Podfile:

```ruby
pod 'EasyAction'
```
